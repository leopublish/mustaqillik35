# Mustaqillik To'yi Festivali Portal

Статический премиальный прототип официального гос-портала для фестиваля к 35-летию независимости.

## Что внутри

- многостраничный портал с разделами `About`, `Events`, `Documents`, `Organization`, `Media`, `Contacts`
- переключение языков `oz / ru / en`
- hero-слайдер, фильтры по карточкам, глобальный поиск
- переключение размера шрифта и контрастного режима
- форма обратной связи с локальным сохранением в `localStorage`
- стилистика, собранная на основе логотипа и логики институционального портала

## Как открыть

1. Откройте [index.html](/C:/Users/hp/OneDrive/Documents/New%20project/index.html) в браузере.
2. Навигация работает без сборщика и сервера.
3. Поиск использует `search.html?q=...`.

## Структура

- `assets/styles/main.css` — дизайн-система и адаптив
- `assets/scripts/data.js` — контент, переводы и модели страниц
- `assets/scripts/site.js` — рендеринг, интерактив, фильтры, поиск, форма
- `docs/production-architecture.md` — как перевести прототип в production на `Next.js + Strapi`

## Важно

Это сильный фронтенд-прототип для презентации, согласования структуры и дальнейшей разработки. Для production-версии нужны:

- `Next.js` frontend
- `Strapi` CMS
- `PostgreSQL`
- файловое хранилище `S3/MinIO`
- реальная обработка форм, загрузок, ролей, SEO и публикации
